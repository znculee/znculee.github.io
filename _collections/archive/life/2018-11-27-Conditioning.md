## push-up

```
vertical-> incline -> kneeling -> horizontal
open-> close
half -> full
two-arms -> uneven-> lever -> one-arm
```

## pull-up

```
vertical -> horizontal -> jackknife -> hanging
bent -> straight
open -> close
static -> half -> full
two-arms -> uneven -> assisted -> one-arm
```
